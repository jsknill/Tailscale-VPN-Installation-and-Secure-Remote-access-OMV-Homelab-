# Tailscale-VPN-Installation-and-Secure-Remote-access-OMV-Homelab-
The goal of this project was to securely access my home lab from anywhere without exposing services directly to the Internet or configuring port forwarding on my router.  My setup now allows me to securely access:  OpenMediaVault (OMV) Nextcloud My home LAN Future services running on the network  using Tailscale.
